How to load static files in Django.

After googling a long time looking to many websites finally I found this way to add static files to Django. As a beginner it's very difficult to debug such errors.

Assuming that you have created a Django project and it has some files.


Step 1:

Open Settings.py in your project directory.

STATICFILES_DIRS = [ os.path.join("add your files path here"), ]

STATICFILES_DIRS is a list of directories which contains paths to your static files. You can use copy path option in properties of folder



Step 2:

Open urls.py in your project directory. Be carefull it is not urls.py in app directory. And add this to urlpatterns

+static(settings.STATIC_URL, document_root=settings.STATIC_ROOT)

Note :Imports must be as shown the image they should not change.

Step 3:

Open HTML page and add this in your first line of code


Step 4:

In your HTML file add this to your src and href tags having path to static files

This works for all CSS,JS and Images


Step 5:

That's it. You're done. Save your file and load the page to see magic
